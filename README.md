## About The Project

Mood tracking program.  
Track your mental and physical health. Log in daily.

<!-- GETTING STARTED -->
## Getting Started

1. Run in your terminal
   ```sh
   python UI.py
   ```

<!-- USAGE EXAMPLES -->
## Usage

1. Log in your tracker or create a new one
2. Add a Log everyday
3. See and Review your progress
